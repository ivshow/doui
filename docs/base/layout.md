# Layout 布局

继承 `Antd Button`的所有 Api

## 基本使用

::: demo

```vue
<template>
  <d-layout>
    <template #sider-header>
      <div class="logo" />
    </template>
  </d-layout>
</template>
```

:::

## 属性

| 参数 | 说明     | 类型   | 默认值    |
| ---- | -------- | ------ | --------- |
| type | 按钮类型 | string | `primary` |
